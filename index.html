Instructions

CloneThis.site is an open source program with the intent of assisting new web developers build their skills by having them recreate web pages that will use all the basics.

This page was created using purely html and css. (Minus this modal which does not need to be re-created.)
I challenge you to re-create it! It will work all of the basic html and css skills, from absolutes, to z-index, this is a great project to start with and make your own!

So open up your developer tools (ctrl-i/command-i), and get started!
